# Fitness tracker

An app that can generate a fitness and nutrition plan using an LLM.

![](screenshot.webp)

## Testing locally

Set the below environment variables:

1. `OPENAI_KEY`: OpenAI API Key
2. `SERPERDEV_API_KEY`: [Serper](https://serper.dev/) key

Run the below command:

```bash
solara run app.py
```
